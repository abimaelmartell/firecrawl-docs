## What's New in v2

### Enhanced Status Tracking
The v2 Extract Status API provides improved monitoring capabilities:

- **Real-time progress**: Better tracking of extraction progress through complex workflows
- **FIRE-1 agent status**: Detailed status reporting for agent-based extractions
- **Enhanced error reporting**: More detailed error messages and debugging information

### Performance Improvements
- **Faster status retrieval**: Optimized status checking for large extraction jobs
- **Better reliability**: Enhanced monitoring for production workloads
- **Improved accuracy**: More precise status reporting for complex extractions

The API parameters and response format remain unchanged from v1, ensuring seamless migration while providing enhanced monitoring capabilities for the advanced v2 extraction features.

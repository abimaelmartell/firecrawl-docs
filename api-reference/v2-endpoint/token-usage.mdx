## What's New in v2

### Enhanced Token Usage Tracking
The v2 Token Usage API provides improved visibility into advanced Extract feature costs:

- **FIRE-1 agent usage**: Track token consumption for intelligent agent operations
- **Search layer costs**: Monitor tokens used for URL-free extraction operations
- **Complex schema processing**: Track token usage for advanced JSON schema extractions
- **Multi-URL extraction**: Enhanced tracking for batch extraction operations

### Advanced Extract Analytics
- **Agent interaction costs**: Monitor token usage for complex web interactions
- **Dynamic content processing**: Track additional tokens for JavaScript-heavy site extractions
- **Enhanced model usage**: Monitor token consumption for improved extraction accuracy
- **Pagination handling**: Track tokens used for intelligent pagination operations

### Performance Improvements
- **Faster retrieval**: Optimized token usage data access for complex Extract operations
- **Better categorization**: Enhanced cost breakdown for advanced v2 Extract features
- **Improved accuracy**: More precise token tracking for FIRE-1 agent operations

The API maintains compatibility with v1 while providing enhanced token tracking for all the advanced v2 Extract capabilities and their associated token consumption.

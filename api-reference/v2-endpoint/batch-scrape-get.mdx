## What's New in v2

### Enhanced Batch Status Tracking
The v2 Batch Scrape Status API provides improved monitoring for advanced batch operations:

- **Format-specific progress**: Track progress of complex format configurations across the batch
- **Parser operation status**: Monitor PDF and other parser operations across all URLs
- **Proxy mode tracking**: Status reporting for different proxy modes used in the batch
- **Location-based monitoring**: Track progress of country/language-specific operations

### Advanced Batch Analytics
- **Change tracking progress**: Monitor change detection operations across batch URLs
- **Browser automation status**: Track progress of enhanced action operations
- **Error categorization**: Better error reporting for complex batch scenarios
- **Resource usage tracking**: Monitor credit and resource consumption for advanced features

### Performance Improvements
- **Faster status retrieval**: Optimized for large batch jobs with complex configurations
- **Better progress tracking**: Enhanced monitoring for advanced v2 features in batch operations
- **Improved reliability**: Better status reporting for production batch workloads

The API maintains compatibility with v1 while providing enhanced monitoring for all the advanced v2 features applied to batch operations.

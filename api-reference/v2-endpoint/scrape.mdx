## What's New in v2

### Advanced Format System
v2 introduces a sophisticated format system with type-specific configurations:

**Simple Formats:**
```json
{
  "formats": [
    {"type": "markdown"},
    {"type": "html"},
    {"type": "rawHtml"},
    {"type": "links"},
    {"type": "summary"}
  ]
}
```

**Advanced Screenshot Configuration:**
```json
{
  "formats": [
    {
      "type": "screenshot",
      "fullPage": true,
      "quality": 90,
      "viewport": {
        "width": 1920,
        "height": 1080
      }
    }
  ]
}
```

**JSON Extraction with Schema:**
```json
{
  "formats": [
    {
      "type": "json",
      "schema": {
        "type": "object",
        "properties": {
          "title": {"type": "string"},
          "price": {"type": "number"}
        }
      },
      "prompt": "Extract product information"
    }
  ]
}
```

### Enhanced Parser System
**v1**: Simple `parsePDF` boolean
**v2**: Advanced `parsers` array with granular control:

```json
{
  "parsers": ["pdf"]
}
```

When enabled, PDF content is extracted and converted to markdown (1 credit per page). When disabled, PDFs are returned in base64 encoding (1 credit total).

### Advanced Change Tracking
```json
{
  "formats": [
    {
      "type": "changeTracking",
      "modes": ["git-diff", "json"],
      "schema": {
        "type": "object",
        "properties": {
          "title": {"type": "string"}
        }
      },
      "tag": "production"
    }
  ]
}
```

### Enhanced Scraping Options
**New in v2:**
- `removeBase64Images`: Removes overwhelming base64 image data (default: true)
- `blockAds`: Enhanced ad-blocking and cookie popup blocking (default: true)
- `location`: Country/language-specific proxying with timezone emulation

### Browser Automation Actions
Enhanced action system with new capabilities:
- **executeJavascript**: Run custom JavaScript on pages
- **pdf**: Generate PDFs with format/orientation options
- **Enhanced screenshot**: Viewport configuration and quality control
- **Advanced interactions**: Click, write, scroll, press keys with improved selectors

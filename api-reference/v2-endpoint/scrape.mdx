## What's New in v2

### Better Output Formats
Configure exactly how you want your data:

**Basic formats:**
```json
{
  "formats": [
    {"type": "markdown"},
    {"type": "html"},
    {"type": "screenshot"}
  ]
}
```

**Custom screenshots:**
```json
{
  "formats": [
    {
      "type": "screenshot",
      "fullPage": true,
      "quality": 90,
      "viewport": {"width": 1920, "height": 1080}
    }
  ]
}
```

**Extract structured data:**
```json
{
  "formats": [
    {
      "type": "json",
      "schema": {
        "type": "object",
        "properties": {
          "title": {"type": "string"},
          "price": {"type": "number"}
        }
      }
    }
  ]
}
```

### PDF Handling
v1: simple on/off switch. v2: full control:

```json
{
  "parsers": ["pdf"]
}
```

On: clean markdown from PDFs (1 credit per page). Off: raw PDF file (1 credit total).

### Track Changes
See what changed since your last scrape:

```json
{
  "formats": [
    {
      "type": "changeTracking",
      "modes": ["git-diff"],
      "tag": "production"
    }
  ]
}
```

### New Options
- `removeBase64Images` - Skip heavy image data (default: on)
- `blockAds` - Better ad blocking and popup handling (default: on)  
- `location` - Scrape from different countries/languages

### Do More on Pages
- Run custom JavaScript
- Generate PDFs with custom settings
- Better clicking, typing, scrolling

## What's New in v2

### Better Output Formats
You can now configure exactly how you want your data:

**Basic formats:**
```json
{
  "formats": [
    {"type": "markdown"},
    {"type": "html"},
    {"type": "screenshot"}
  ]
}
```

**Custom screenshots:**
```json
{
  "formats": [
    {
      "type": "screenshot",
      "fullPage": true,
      "quality": 90,
      "viewport": {"width": 1920, "height": 1080}
    }
  ]
}
```

**Extract structured data:**
```json
{
  "formats": [
    {
      "type": "json",
      "schema": {
        "type": "object",
        "properties": {
          "title": {"type": "string"},
          "price": {"type": "number"}
        }
      }
    }
  ]
}
```

### PDF Handling
v1 had a simple on/off switch for PDFs. v2 gives you control:

```json
{
  "parsers": ["pdf"]
}
```

Turn it on to get clean markdown from PDFs (costs 1 credit per page). Turn it off to get the raw PDF file (1 credit total).

### Track Changes
See what changed on a page since your last scrape:

```json
{
  "formats": [
    {
      "type": "changeTracking",
      "modes": ["git-diff"],
      "tag": "production"
    }
  ]
}
```

### New Options
- `removeBase64Images` - Skip heavy image data (on by default)
- `blockAds` - Better ad blocking and popup handling (on by default)  
- `location` - Scrape from different countries/languages

### Do More on Pages
- Run custom JavaScript
- Generate PDFs with custom settings
- Better clicking, typing, and scrolling

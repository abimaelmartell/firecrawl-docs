## What's New in v2

### Natural Language Crawler Configuration
The v2 Crawl API introduces a `prompt` parameter that allows you to configure crawler options using natural language:

```json
{
  "url": "https://example.com",
  "prompt": "Crawl only the blog posts and documentation pages, ignore marketing pages"
}
```

The prompt generates crawler options automatically, though explicitly set parameters will override the generated equivalents.

### Simplified Sitemap Handling
- **v1**: Used `ignoreSitemap` boolean parameter
- **v2**: Uses `sitemap` enum with values `"skip"` or `"include"` (default: `"include"`)

### Removed Parameters
- `maxDepth`: No longer available in v2. Use `maxDiscoveryDepth` for depth-based crawling control.

## What's New in v2

### Better Error Messages
- Clear errors when your natural language prompts don't work
- Specific messages for sitemap and depth limit issues
- Better errors for domain-wide crawling problems

### More Error Details
- Detailed errors for format processing issues
- Specific messages for PDF and parser failures
- Better debugging for location-based crawling
- Clear errors for change tracking problems

### Faster and Easier Debugging
- Quicker error retrieval for large crawls
- Better error grouping
- More helpful error context

Same API as v1, just with much better error reporting for all the new v2 features.

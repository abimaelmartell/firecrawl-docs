## What's New in v2

The v2 Crawl Errors API maintains the same functionality as v1 with enhanced error reporting:

### Enhanced Error Reporting
- Improved error categorization and detailed error messages
- Better debugging information and troubleshooting guidance
- Enhanced error filtering and pagination capabilities

The API parameters and response format remain unchanged from v1, ensuring seamless migration while providing more comprehensive error information.

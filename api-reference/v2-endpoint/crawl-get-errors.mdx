## What's New in v2

### Enhanced Error Reporting
The v2 Crawl Errors API provides improved debugging for advanced crawling features:

- **Prompt configuration errors**: Detailed error reporting for natural language prompt parsing
- **Sitemap mode errors**: Specific error messages for different sitemap handling modes
- **Discovery depth issues**: Error tracking for `maxDiscoveryDepth` limitations
- **Domain crawling errors**: Enhanced error reporting for `crawlEntireDomain` operations

### Advanced Error Categories
- **Format processing errors**: Detailed errors for complex format configurations
- **Parser operation failures**: Specific error messages for PDF and other parser issues
- **Location-based errors**: Error reporting for country/language-specific proxy issues
- **Change tracking errors**: Debugging information for change detection operations

### Performance Improvements
- **Faster error retrieval**: Optimized for large crawl jobs with complex configurations
- **Better categorization**: Enhanced error grouping for easier debugging
- **Improved debugging**: More detailed error context for advanced v2 features

The API maintains compatibility with v1 while providing enhanced error reporting for all the advanced v2 crawling capabilities.

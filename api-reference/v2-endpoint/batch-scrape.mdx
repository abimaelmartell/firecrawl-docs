## What's New in v2

### All the New Formats Work in Batches
- Use custom screenshots, JSON extraction, and change tracking across all URLs
- Process PDFs in batches with the new parser system
- Apply the same format settings to every URL in your batch

### Better Batch Processing
- Scrape from different countries/languages for entire batches
- Use stealth, basic, or auto proxy modes across all URLs
- All the new scraping options work in batch mode

### Faster and More Reliable
- Better resource management for complex batches
- Improved progress tracking
- Better error handling

Same API as v1, but now you get all the new v2 features across your entire batch.

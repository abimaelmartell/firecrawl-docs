## What's New in v2

The v2 Extract API maintains the same request and response structure as v1, but includes significant improvements:

### Enhanced Performance & Reliability
- Improved extraction accuracy and speed
- Better handling of complex page structures
- Enhanced support for dynamic content

### FIRE-1 Agent Integration
v2 Extract includes enhanced capabilities for handling pagination and complex interactions through improved agent-based processing, enabling more comprehensive data extraction from dynamic websites.

The API parameters and response format remain unchanged from v1, ensuring easy migration while providing better results.
